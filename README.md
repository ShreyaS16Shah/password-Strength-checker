# Password Strength Checker

## Description
This project is a simple Python-based cybersecurity tool that checks the strength of a user-entered password using common security rules.

## Features
- Checks minimum password length
- Verifies presence of uppercase letters
- Verifies presence of lowercase letters
- Checks for digits
- Checks for special characters

## Technologies Used
- Python
- Regular Expressions (re module)

## How to Run
1. Clone the repository
2. Run `python password_checker.py`
3. Enter a password when prompted

## Use Case
This project demonstrates basic authentication security principles and helps prevent weak password usage.
